# Premium Secure Custom YouTube Player 🎬

مشغل فيديو احترافي يعتمد على `YouTube IFrame API`، بواجهة مخصصة بالكامل باستخدام تقنية `Glassmorphism` وبنية `Vanilla JavaScript` خالية من المكتبات الخارجية.

## الميزات (Features)
- 🔒 **حماية الـ UX**: منع النقر الأيمن، تعطيل اختصارات أدوات المطور لمنع النسخ البسيط.
- 🎨 **تصميم عصري**: واجهة مستخدم مشابهة لـ Netflix/Apple TV.
- ⚡ **أداء عالي**: استهلاك منخفض للذاكرة باستخدام `requestAnimationFrame` وتحديثات DOM الدقيقة.
- 🧠 **الذاكرة (Resume)**: حفظ تقدم الفيديو محلياً (`localStorage`) لاستكمال المشاهدة.
- ⌨️ **اختصارات الكيبورد**: (Space للتشغيل، F لملء الشاشة، M لكتم الصوت، والأسهم للتقديم والتأخير).

## الاستخدام (Usage)
لا يتطلب المشروع أي عملية بناء (No Build Tools). فقط قم برفع الملفات إلى أي خادم (مثل Apache, Nginx, أو Vercel/Netlify) أو افتح `index.html` باستخدام `Live Server` في VSCode.

**ملاحظة:** بسبب سياسات أمان المتصفح (`CORS` و `IFrame Restrictions`)، بعض الخصائص (مثل حفظ الوقت محلياً أو تشغيل الفيديو تلقائياً) قد لا تعمل إذا فتحت الملف مباشرة كـ `file://`، لذا يُرجى تشغيله من خلال Local Server.
