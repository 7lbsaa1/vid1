# 🎬 منصة حلبسة - المشغل الاحترافي (7lbsaa Premium Player)

مشغل فيديو متطور ومخصص بالكامل مبني على واجهة **YouTube Iframe API**، مصمم خصيصاً لمنصة "حلبسة" التعليمية. يهدف هذا المشروع إلى تقديم تجربة مشاهدة آمنة، سلسة، وخالية من المشتتات مع واجهة مستخدم زجاجية (Glassmorphism) فائقة الجودة.

## ✨ المميزات الرئيسية (Features)

- 🧠 **نظام حفظ التقدم الذكي:** تتبع وحفظ وقت المشاهدة لكل فيديو بشكل مستقل باستخدام `localStorage`، مما يتيح للطالب استكمال المحاضرة من حيث توقف.
- 🎨 **واجهة زجاجية داكنة (Dark Glassmorphism):** تصميم عصري يبرز عناصر التحكم فوق الفيديو بشكل أنيق ولا يعيق المشاهدة، مع دعم كامل للاتجاه الصحيح (LTR).
- 🖱️ **إخفاء تلقائي ذكي:** يختفي شريط التحكم ومؤشر الماوس تلقائياً عند عدم الحركة لتوفير انغماس كامل في الشرح.
- ⌨️ **اختصارات لوحة مفاتيح متقدمة:** متوافقة تماماً مع المعايير العالمية ليوتيوب:
  - `K` أو `Space` : تشغيل / إيقاف.
  - `J` / `L` أو `الأسهم الجانبية` : تأخير / تقديم 10 ثوانٍ.
  - `الأسهم لأعلى/لأسفل` : التحكم في مستوى الصوت مع تحديث فوري للشريط.
  - `F` : ملء الشاشة.
  - `M` : كتم الصوت.
  - `T` : وضع المسرح.
- 📱 **متجاوب 100% (Responsive):** يتكيف بشكل مثالي مع جميع الشاشات (كمبيوتر، تابلت، هواتف ذكية).
- 🚀 **أداء فائق (60fps):** استخدام `requestAnimationFrame` لتحديث شريط التقدم بسلاسة فائقة دون استهلاك موارد الجهاز.
- 🛡️ **حماية وخصوصية:** إخفاء لوجو يوتيوب، منع الفيديوهات المقترحة الخارجية، وتعطيل الإعلانات المنبثقة داخل المشغل.
